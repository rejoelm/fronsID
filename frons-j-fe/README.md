# FRONS-J (Journal Publisher)

FRONS-J is an institutional journal management system built on Solana. It allows academic entities to launch and manage decentralized journals with sovereign editorial control.

## 🚀 Key Features
- **Institutional Subdomains**: Custom routing for multiple institutional journals.
- **Editorial Dashboard**: Manage submissions, reviewer assignments, and publication decisions.
- **On-Chain Editorial Board**: Verify and register private reviewer pools.
- **Wallet-Centric Submissions**: Gasless submission flows powered by Privy.

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS
- **Smart Contract**: FRONS-J Anchor Program
- **Auth**: Privy Unified Social/Wallet Login

## 🏗 Setup & Local Development
```bash
cd frons-j-fe
npm install
npm run dev
```
