# FRONS NFC Profile Cards

This application serves as the public-facing identity hub for the FRONS ecosystem. It is primarily accessed via physical NFC-enabled FRONS cards held by verified researchers.

## 🚀 Key Features
- **Verified Research Profiles**: Beautifully rendered academic resumes fetching live on-chain credentials.
- **Publication History**: Dynamically displays published manuscripts from both the Fronsciers Library and FRONS-J journals.
- **Soulbound Badges**: Visualizes verifiable academic achievements and institutional certifications.

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **Icons**: Lucide React
- **Styling**: Tailwind CSS
- **Integration**: On-chain data fetching from Solana

## 🏗 Setup & Local Development
```bash
cd frons-card-main
npm install
npm run dev
```
