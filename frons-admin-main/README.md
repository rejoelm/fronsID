# FRONS Master Admin Console

This isolated Next.js application serves as the central nervous system for the FRONS ecosystem. It is designed For global administrators to manage protocol integrity, monitor health, and seed critical evidence.

## 🚀 Key Features
- **Ecosystem Analytics**: Real-time aggregation of active users, Walrus blobs, and DOCI NFT minting stats.
- **Vendor API Management**: Secure UI for managing protocol configurations and vendor credentials (Railway, Supabase, Vercel).
- **AI Evidence Seeding**: Direct PDF upload to Walrus, bypassing the public peer-review gate to ingest reliable primary sources into the AI model.

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **Styling**: Tailwind CSS / Shadcn UI
- **Backend**: Supabase
- **Chain**: Solana / Walrus Network

## 🏗 Setup & local Development
```bash
cd frons-admin-main
npm install
npm run dev # Runs on port 3004
```
