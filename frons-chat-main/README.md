# FRONS Vault & AI Chat

This advanced application provides secure storage and intelligent context-aware research assistance for the FRONS ecosystem.

## 🚀 Key Features
- **My Vault**: Client-side AES-256-GCM encryption for private research datasets.
- **Walrus Integration**: Decentralized storage for encrypted blobs, ensuring permanent availability.
- **Evidence Cascade AI**: Intelligent chat interface that cites both public on-chain research and your private vault data.
- **Unified Auth**: Gasless onboarding via Privy with support for social and wallet logins.

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **Cryptography**: Web Crypto API (AES-256)
- **AI**: Integrated Evidence Cascade context model
- **Storage**: Walrus Testnet

## 🏗 Setup & Local Development
```bash
cd frons-chat-main
npm install
npm run dev
```
