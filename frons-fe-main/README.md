# Fronsciers Library

The Fronsciers Library is the primary decentralized publication gateway for the FRONS ecosystem. It empowers independent researchers to publish their work with true cryptographic ownership.

## 🚀 Key Features
- **Manuscript Submission**: Unified flow for uploading research to the blockchain.
- **My Publications**: Dashboard visualizing Metaplex DOCI NFTs ensuring verifiable ownership.
- **On-Chain Peer Review**: Real-time tracking of review states and scores.
- **Author Dashboard**: Centralized hub for managing research output and citations.

## 🛠 Tech Stack
- **Framework**: Next.js 15
- **SDK**: Metaplex Foundation JS
- **Auth**: Privy Unified Social/Wallet Login
- **Styling**: Vanilla CSS / Tailwind Hybrid

## 🏗 Setup & Local Development
```bash
cd frons-fe-main
npm install
npm run dev
```
