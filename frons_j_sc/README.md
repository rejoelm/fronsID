# FRONS-J Smart Contract (B2B)

This Anchor program enables **Institutional Journal Management** on Solana. It allows universities and editorial boards to launch their own branded journals with sovereign control over reviews and publishing workflows.

## 🚀 Key Features
- **Institutional Onboarding**: Supports private editorial board registration and verification.
- **B2B Submission Flow**: Handles submissions specifically for institutional review cycles.
- **Journal State Management**: Tracks journal metadata, issues, and peer-review history independently from the B2C library.

## 🛠 Tech Stack
- **Language**: Rust
- **Framework**: Anchor
- **Storage**: Walrus Testnet Integration

## 🏗 Setup & Build
```bash
anchor build
```
